| **Model Name**                 | **Purpose / What It Extracts**                                                                                  |
| ------------------------------ | --------------------------------------------------------------------------------------------------------------- |
| `prebuilt-invoice`             | Vendor, customer, totals, tax, line items, invoice ID, date, etc.                                               |
| `prebuilt-receipt`             | Merchant name, transaction date/time, total, taxes, items purchased, etc.                                       |
| `prebuilt-businessCard`        | Contact info: name, company, email, phone, address, website, etc.                                               |
| `prebuilt-identityDocument`    | ID type, name, DOB, nationality, document number, expiration date. Works for passports, driver’s licenses, etc. |
| `prebuilt-tax.us.w2`           | Extracts US W-2 tax form fields (wages, employer, employee info).                                               |
| `prebuilt-bankStatement`       | Account number, balance, transactions, bank name, etc.                                                          |
| `prebuilt-creditCard`          | Card type, number, expiry, cardholder name (masked for security).                                               |
| `prebuilt-contract`            | Parties involved, dates, clauses, signatures, key terms.                                                        |
| `prebuilt-healthInsuranceCard` | Member name, policy number, provider info, effective date, etc.                                                 |
| `prebuilt-form`                | Generic structured data — good for semi-structured forms and tables.                                            |
| `prebuilt-read`                | Text extraction (OCR only), including text lines, words, bounding boxes, and language.                          |
| `prebuilt-layout`              | Layout + OCR — paragraphs, tables, checkboxes, selection marks, and reading order.                              |
| `prebuilt-document`            | General-purpose extraction: key-value pairs, entities, and structure for any document.                          |
